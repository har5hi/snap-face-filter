✨ Snap with Filters

SnapAura is a real-time face filter web application built using JavaScript and MediaPipe.

🚀 Features
- 🎭 Multiple face filters (glasses, crown, dog ears, hearts, etc.)
- 🧠 Real-time face tracking using MediaPipe
- 📸 Capture and download filtered images
- 🎨 Modern UI with responsive design

🛠️ Tech Stack
- HTML, CSS, JavaScript
- MediaPipe Face Mesh
- Canvas API
